Ciclos en Java

Los ciclos permiten ejecutar un bloque de código varias veces. Se usan cuando necesitas repetir una acción sin escribir el mismo código muchas veces.

Ciclo FOR

for(int i = 1; i <= 5; i++){
  System.out.println(i);
}

Se usa cuando sabes cuántas veces se repetirá.

Ciclo WHILE

int i = 1;
while(i <= 5){
  System.out.println(i);
  i++;
}

Se usa cuando NO sabes cuántas repeticiones habrá.
